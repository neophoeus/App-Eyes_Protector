# App-Eyes Protector

一款極輕量、零依賴的 Windows 沉浸式護眼助理。

## 💡 核心亮點

1. **無感常駐，溫柔提醒**：平時完全隱形於背景（無系統列圖示），每 20 分鐘以置中的高雅卡片提醒休息，不強暴中斷您的工作心流。
2. **算繪美學，沉浸放鬆**：捨棄傳統呆板的靜態圖，運用純數學運算生成 3D 落葉與微風動畫，打造 20 秒「心跳跟著畫面慢下來」的呼吸時刻。
3. **綠色輕量，隨開即用**：零外部套件依賴、免安裝。由一支純粹呼叫 Windows 原生 API 的 Python 單一執行檔 (.exe) 與簡易的開關腳本構成。

## ⚙️ 技術原理

* **環境偵測**：透過直接呼叫 Windows 底層原生 API (`ctypes.windll.shell32.SHQueryUserNotificationState`)，智慧判斷使用者是否處於全螢幕遊戲、看影片或休眠狀態，避免干擾。
* **渲染引擎**：零圖檔依賴，完全使用 Python `tkinter.Canvas` 原生繪圖工具。結合數學幾何（三角函數與隨機變數）建構落葉多邊形，即時運算產生 3D 翻轉與微風飄落的物理動態效果。
* **極輕量部署**：捨棄所有外部第三方套件（如 Pillow、Pystray），並透過 PyInstaller 打包為單一背景隱形執行檔 (`.exe`)，搭配 BAT 批次腳本達成簡易的開機自動啟動與卸載。

## 🚀 快速上手

### 取得執行檔

本專案已透過 PyInstaller 編譯為單一免安裝執行檔。您可以直接在 `dist` 資料夾中找到 `main.exe`。

### 設定開機自動啟動

* **開啟自動保護**：點擊兩下 `setup_startup.bat`，程式將會在背景默默開始保護您的雙眼。
* **取消自動保護**：點擊兩下 `remove_startup.bat` 即可輕鬆解除。

### 手動測試與退出

* **測試動畫效果**：您可以隨時透過命令提示字元執行 `main.exe --test` 來立即測試 10 秒提醒與 5 秒休息。
* **手動退出**：當休息畫面的樹葉飄落時，點擊畫面右上角的 `✕` 符號即可完全關閉背景程式。

## 🛠️ 開發與編譯

如果您想自行修改原始碼並重新編譯，請確認您已安裝 Python 3，並執行以下指令：

```bash
pip install pyinstaller
python -m PyInstaller --noconsole --onefile --noconfirm main.py
```

編譯完成後，新的執行檔將會產生在 `dist` 目錄下。
